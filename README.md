# `golembic-blend`

> SQL Schema Management in Go, inspired by `sqlalchemy/alembic`

[![GoDoc][1]][2]

The goal of this repository is to reframe `github.com/dhermes/golembic` in
terms of the primitives in `github.com/blend/go-sdk/db`.
